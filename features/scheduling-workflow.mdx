---
title: "Scheduling a Workflow"
---


1. Navigate to "My Workflows"Toggle the **"Schedule Workflow"** option  (screenshot)
2. Choose frequency:  (screenshot)
   - Daily
   - Weekly (select day)
   - Every 15/30/60 days
3. Select target ASINs: (screenshot)
   - Paste ASINs, or
   - Filter ASINs by top X% by sales

#### **Best Practices for Scheduling**

- Run PPC workflows weekly or every 2 weeks
- Run listing hygiene workflows daily
- Run listing content workflows monthly
- Run inventory workflows weekly
